# Crypto Tracker App

A Crytpo tracker app is build with provider and shared preference in flutter. It has a feature of registering and login user which is done with the help of Firebase. It has a feature of dark mode and light mode theme. It also has a feature of adding any crypto to favourite. For the data, "coingeko" api is used.
